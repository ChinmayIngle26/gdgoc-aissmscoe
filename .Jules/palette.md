## 2024-05-19 - Missing ARIA Labels on Interactive Icons
**Learning:** Several interactive elements (such as the main navigation hamburger menu, floating social links, and team member profile links) were missing `aria-label` attributes or had incorrectly copied attributes (e.g., LinkedIn button showing "Join Whatsapp Group"). This causes major issues for screen reader accessibility since the elements contain only visual icons without text content.
**Action:** When adding or copying icon-only buttons or interactive links (like social icons or custom menus), always ensure a specific, context-appropriate `aria-label` is applied, and update any copied strings immediately. Use dynamic ARIA states (like `aria-expanded` and `aria-label` based on toggle state) for interactive components like hamburger menus.
